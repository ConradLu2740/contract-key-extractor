# 非技术人员安装指南

## 第一步：安装必要软件

### 1. 安装Python

1. 打开浏览器，访问 https://www.python.org/downloads/
2. 点击 "Download Python 3.x.x" 按钮（下载最新版本）
3. 下载完成后，双击运行安装程序
4. **重要**：勾选 "Add Python to PATH" 选项
5. 点击 "Install Now" 安装
6. 安装完成后，关闭安装程序

### 2. 安装Go

1. 打开浏览器，访问 https://go.dev/dl/
2. 下载Windows版本（如 go1.21.x.windows-amd64.msi）
3. 双击运行安装程序
4. 一路点击 "Next" 完成安装
5. 安装完成后，关闭安装程序

### 3. 安装Node.js

1. 打开浏览器，访问 https://nodejs.org/
2. 下载LTS版本（长期支持版）
3. 双击运行安装程序
4. 一路点击 "Next" 完成安装
5. 安装完成后，关闭安装程序

---

## 第二步：获取智谱AI API Key

1. 打开浏览器，访问 https://open.bigmodel.cn/
2. 注册并登录账号
3. 进入控制台，找到API Key管理
4. 创建新的API Key并复制保存

---

## 第三步：配置项目

1. 打开项目文件夹 `contract-key-extractor`
2. 进入 `ai-service` 文件夹
3. 找到 `.env.example` 文件，复制一份
4. 将复制的文件重命名为 `.env`
5. 用记事本打开 `.env` 文件
6. 将 `your_api_key_here` 替换为你的API Key
7. 保存并关闭文件

---

## 第四步：启动项目

1. 双击运行 `start.bat` 文件
2. 等待安装依赖和启动服务（首次启动可能需要几分钟）
3. 看到浏览器自动打开项目页面
4. 开始使用！

---

## 常见问题

### Q: 双击start.bat后闪退？

A: 可能是软件未正确安装。请按照第一步重新安装，确保安装时勾选了正确的选项。

### Q: 提示端口被占用？

A: 可能是之前启动的服务还在运行。双击运行 `stop.bat` 停止所有服务，然后重新运行 `start.bat`。

### Q: 提取结果为空？

A: 请检查：
1. API Key是否正确配置
2. 网络是否正常
3. 文件是否损坏

### Q: 如何停止服务？

A: 双击运行 `stop.bat` 文件即可停止所有服务。
